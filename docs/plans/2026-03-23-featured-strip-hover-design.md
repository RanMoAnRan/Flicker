# 首页精选片单 Hover 视觉设计

## 背景

首页“精选片单”当前 hover 反馈较弱，只继承了通用卡片的轻微上浮效果，缺少更明确但仍然克制的悬停反馈。

## 目标

- 仅增强首页“精选片单”的桌面端 hover 体验
- 保持视觉克制，不引入夸张动效
- 让用户感受到卡片被轻轻提起、海报被点亮

## 方案

采用“克制高级”风格：

- 卡片整体轻微上浮
- 海报轻微放大
- 阴影略加深
- 标题轻微提亮，向品牌暖色靠近
- badge 对比度略微增强

## 实现边界

- 仅作用于首页精选片单
- 仅在支持 hover 的桌面设备启用
- 不改结构，不改移动端样式

## 风险与验证

- 风险：动效过重会让六列卡片显得吵
- 对策：严格控制位移、缩放和阴影幅度
- 验证：
  - 鼠标悬停时卡片反馈更清晰
  - 整排卡片同时存在时仍然克制
  - 移动端与其他卡片不受影响
