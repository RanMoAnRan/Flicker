# 浮光影视门户化改版设计

## 目标

将现有的轻量影视站改造成更像完整影视网站的界面，同时保持当前 CMS 采集接口可直接驱动，不引入额外后端依赖。

## 核心方向

- 首页采用“影视门户 + 资源站检索”的混合模式
- 详情页强调海报、标签、剧情和线路切换
- 播放页强化沉浸感，同时保留高效选集
- 设置页升级为统一风格的片源管理中心

## 真实可用维度

- 分类：`class`、`type_id`、`type_name`
- 搜索：`wd`
- 分页：`pg`
- 更新时间：`vod_time`
- 更新状态：`vod_remarks`
- 播放线路：`vod_play_from`、`vod_play_url`
- 详情信息：海报、导演、演员、简介

## 页面结构

### 首页

- 首屏推荐区
- 分类频道导航
- 状态筛选条
- 精选片单区
- 海报网格列表

### 详情页

- 海报与基础资料主舞台
- 剧情简介区
- 线路切换与剧集区

### 播放页

- 大播放器主舞台
- 当前播放信息侧栏
- 线路与选集控制台

### 设置页

- 片源管理介绍区
- 数据源添加表单
- 已配置片源列表

## 视觉语言

- 深色影院氛围背景
- 金橙与冷蓝双重高光
- 面板化布局与柔和发光边缘
- 片单、线路、状态统一为影视站风格标签系统

## 实现说明

- 保留原有多页面架构
- 公共元数据识别逻辑集中在 `js/app.js`
- 统一样式集中在 `css/style.css`
- 不引入框架和构建链
