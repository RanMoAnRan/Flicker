# 首页精选片单比例微调设计

## 背景

首页“精选片单”区域当前使用纵向卡片，视觉上相比“今日焦点”的四张卡片，下半部分文字区占比略大，导致海报区域离卡片底部显得不够近，整体比例不够统一。

## 目标

- 保持“精选片单”现有纵向卡片结构不变
- 让六张精选卡在桌面端的视觉比例更接近“今日焦点”
- 不改移动端横向滚动布局

## 方案

采用“海报略增高 + 下半部分收紧”的轻量方案：

- 仅调整桌面端基础样式
- 小幅提升 `.featured-poster` 高度
- 收紧卡片上下分区之间的 `gap`
- 收紧文字区内部的间距与行高

## 风险与验证

- 风险：若海报过高，六列卡片可能显得偏窄偏高
- 对策：只做小幅调整，不改变列数和结构
- 验证：
  - 桌面端六张精选卡更接近“今日焦点”的紧凑比例
  - 移动端保持现有表现
